# Web Ciphers Toolkit

Interactive web application for encrypting and decrypting text using classic historical ciphers. All cryptographic operations are performed client-side

## Included Ciphers

- **Vigenère Cipher:** Polyalphabetic substitution cipher using a keyword.
- **Caesar Cipher:** Classic substitution cipher with a numeric shift.
- **Affine Cipher:** Mathematical cipher using a multiplier (coprime to 26) and a shift.
- **Playfair Cipher:** Digraph substitution cipher using a dynamic 5x5 key matrix.
